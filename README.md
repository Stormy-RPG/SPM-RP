# Description 

A resource pack from SPm players - for SPm players and others!

Combines content from multiple historical server resource packs: **MRP**, **MIRRP**, **Beer Weeks** and others - adapted for current Minecraft versions. Includes:

- **Drinks & Medications** - potions and medicines with custom textures and names
- **Food** - custom food items
- **Gambling** - poker chips, playing cards and other gambling items
- **Decor, Logos & More** - items for RP location design
- **Ghost Armor** - custom ghost armor with player body display (RPT is required)
- **Plushies** - player character plush toys and collectibles
- **Seasonal Content** - Halloween sweets, Christmas items
- **Tools & Scrap** - packed tools, scrap metal and more

The full item list is available in the [document](https://docs.google.com/document/d/1wawDhK3q0TYNNQSxZZM0KMY7ksTF0p1AE2gV5J8gag0/edit?usp=sharing).

> ⚠️ **Support for version 1.21.1 has been dropped.** Supported versions: **1.21.7 - 1.21.11**.
